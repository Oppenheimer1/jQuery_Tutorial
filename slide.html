To slideDown and slideUp the element when the site loads automatically:

The jQuery code to for this example is:

$(document).ready(function(){
    $(".slide1").slideDown(1000,"linear").slideUp(1000,"linear"); 
});
This one above auto Slides Down and then Slides Up while the one below you need to click:

The css code to for this example is:

.slide1, .up1{
    padding: 5px;
    text-align: center;
    background-color: #A0CFEC;
    border: solid 1px #c3c3c3;
}
.slide1 {
    padding: 50px;
    display: none;
}

The html code to for this example is:

  
  <div class="up1">Click to slide down panel</div>
  <div class="slide1">Fun Times</div>

To Just Slide Down an element click the panel or click the button


The jQuery code to slide down an element is:

$(document).ready(function(){
    $(".up2").click(function(){
        $(".slide2").slideDown(1000,"linear");
    });
});
See how you have to add a click function an apply it to the button class up2a

The css code to do this is:

.slide2, .up2{
    padding: 5px;
    text-align: center;
    background-color: #A0CFEC;
    border: solid 1px #c3c3c3;
}
.slide2 {
    padding: 50px;
    display: none;
}

The html code to do this is:


      <div class="up2">Click to slide down panel</div>
      <div class="slide2">Fun Times</div>
      <button class="up2">Slide Down</button></br>

To toggle the element click the panel or click the button


The jQuery code to toggle and element between slide down and slide up is:

$(document).ready(function(){
    $(".up4").click(function(){
        $(".slide4").slideToggle(1000,"linear");
    });
});
See how you have to add a click function an apply it to the button class up2a

The css code used for this is:

.slide4, .up4{
    padding: 5px;
    text-align: center;
    background-color: #A0CFEC;
    border: solid 1px #c3c3c3;
}
.slide4 {
    padding: 50px;
    display: none;
}

The html code used for this is:


  <div class="up4">Click to slide up or down the panel</div>
  <div class="slide4">Fun Times</div>
  <div>
  <button class="up4">Toggle</button></br>
