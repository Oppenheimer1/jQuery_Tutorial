To FadeIn and FadeOut the element when the site loads automatically:










The jQuery code to for this example is:

$(document).ready(function(){
$(".target5").fadeOut(1000, "linear").fadeIn(1000, "linear");
});
This one above auto Fades In and then Fades Out while the one below you need to click:

The css code to for this example is:

.target5 {
  color: rgb(255, 255, 255); 
  width: 150px; height: 
  100px; float: left; 
  border: 1px solid rgb(34, 34, 34); 
  margin: 10px; 
  padding: 5px; 
  opacity: 1; 
  display: block; 
  background: rgb(143, 188, 143);
}

The html code to for this example is:


  <div class="target5">
  .target
</div>

To Just Fade Out an element








The jQuery code to fade out an element is:

$(document).ready(function(){
    $(".target6a").click(function(){
        $(".target6").fadeOut(1000, "linear");
    });
});
See how you have to add a click function an apply it to the button class target6a

The css code used is:

.target6 {
  color: rgb(255, 255, 255); 
  width: 150px; height: 
  100px; float: left; 
  border: 1px solid rgb(34, 34, 34); 
  margin: 10px; 
  padding: 5px; 
  opacity: 1; 
  display: block; 
  background: rgb(143, 188, 143);
}

The html code used is:


<div class="target6">
  .target
</div>
<button class="target6a">Fade Out</button></br>
