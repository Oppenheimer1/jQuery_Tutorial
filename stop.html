Click to slide down panel
Hello world!

The jQuery code to start slide and then to stop an element from sliding is

$(document).ready(function(){
    $(".animate7a").click(function(){
        $(".animate7").slideDown(5000);
    });
    $(".stop").click(function(){
        $(".animate7").stop();
    });
});
See how you have to add a click function an apply it to the button class target2a

The css code used is:

.animate7, .animate7a {
    padding: 5px;
    font-size: 18px;
    text-align: center;
    background-color: lightblue;
    color: white;
    border: solid 1px #666;
    border-radius: 3px;
}
.animate7 {
    padding: 50px;
    display: none;
}

The html code used is:

  
<button class="stop">Stop sliding</button>

<div class="animate7a">Click to slide down panel</div>
<div class="animate7">Hello world!</div>
