To hide and show the element when the site loads automatically:










The jQuery code to for this example is:

$(document).ready(function(){
$(".target").hide(1000, "linear").show(1000, "linear");
});
This one above auto loads while the one below you need to click:

The css code to for this example is:

To Just Hide an element








The jQuery code to hide an element is:

$(document).ready(function(){
    $(".target2a").click(function(){
        $(".target2").hide(1000, "linear");
    });
});
See how you have to add a click function an apply it to the button class target2a

The css code used is:

.target2 {
  color: rgb(255, 255, 255); 
  width: 150px; height: 
  100px; float: left; 
  border: 1px solid rgb(34, 34, 34); 
  margin: 10px; 
  padding: 5px; 
  opacity: 1; 
  display: block; 
  background: rgb(143, 188, 143);
}

The html code used is:

  
<div class="target2">
  .target
</div>
<button class="target2a">Hide</button></br>

To Toggle an element, This enables you to hide or show the element each time you click toggle








The jQuery code to toggle an element is:

$(document).ready(function(){
    $(".target4a").click(function(){
        $(".target4").toggle(1000, "linear");
    });
});
See how you have to add a click function an apply it to the button class target4a

The css code used is:

.target4 {
  color: rgb(255, 255, 255); 
  width: 150px; height: 
  100px; float: left; 
  border: 1px solid rgb(34, 34, 34); 
  margin: 10px; 
  padding: 5px; 
  opacity: 1; 
  display: block; 
  background: rgb(143, 188, 143);
}

The html code used is:


<div class="target4">
  .target
</div>
<button class="target4a">Toggle</button></br>
